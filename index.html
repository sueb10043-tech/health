/** * 혈당 및 체중 매니저 Pro - 스프레드시트 연동 최종본 * 스프레드시트 컬럼 인덱스 (image_3cd69d.png 기준): * 0:A(날짜), 1:B(아침혈당), 2:C(아침메모), 3:D(점심식단), 4:E(점심혈당), 5:F(저녁식단), 6:G(저녁혈당), 7:H(기타메모), 8:I(체중), 9:J(걸음), 10:K(성공) */ function doGet() { const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); const data = sheet.getDataRange().getValues(); const records = {}; for (let i = 1; i < data.length; i++) { const row = data[i]; if (!row[0]) continue; try { let dateObj = row[0] instanceof Date ? row[0] : new Date(row[0]); // 날짜 오차 보정을 위해 12시간을 더한 뒤 yyyy-MM-dd로 변환 const dateKey = Utilities.formatDate(new Date(dateObj.getTime() + (12 * 60 * 60 * 1000)), "GMT+9", "yyyy-MM-dd"); records[dateKey] = { date: dateKey, morning_sugar: row[1] || "", // B열: 아침 혈당 morning_memo: row[2] || "", // C열: 아침 메모 lunch_menu: row[3] || "", // D열: 점심 식단 (메뉴) lunch_sugar: row[4] || "", // E열: 점심 혈당 dinner_menu: row[5] || "", // F열: 저녁 식단 (메뉴) dinner_sugar: row[6] || "", // G열: 저녁 혈당 extra_memo: row[7] || "", // H열: 기타 메모 body_weight: row[8] || "", // I열: 체중 step_count: row[9] || "", // J열: 걸음수 is_success: String(row[10]).toUpperCase() === "TRUE" || row[10] === true // K열: 성공 }; } catch (e) { console.error("데이터 읽기 오류: " + e.toString()); } } return ContentService.createTextOutput(JSON.stringify({ records: records })) .setMimeType(ContentService.MimeType.JSON); } function doPost(e) { const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); let data; try { data = JSON.parse(e.postData.contents); } catch (err) { return ContentService.createTextOutput(JSON.stringify({ result: "error" })).setMimeType(ContentService.MimeType.JSON); } const dateStr = data.date; const values = sheet.getDataRange().getValues(); let targetRow = -1; for (let i = 1; i < values.length; i++) { if (values[i][0]) { let rowDate = values[i][0] instanceof Date ? values[i][0] : new Date(values[i][0]); let formattedRowDate = Utilities.formatDate(new Date(rowDate.getTime() + (12 * 60 * 60 * 1000)), "GMT+9", "yyyy-MM-dd"); if (formattedRowDate === dateStr) { targetRow = i + 1; break; } } } // 저장할 데이터 배열 (A~K 컬럼 순서 그대로 유지) const rowData = [ dateStr, data.morning_sugar, data.morning_memo, data.lunch_menu, data.lunch_sugar, data.dinner_menu, data.dinner_sugar, data.extra_memo, data.body_weight, data.step_count, data.is_success ]; if (targetRow > 0) { sheet.getRange(targetRow, 1, 1, rowData.length).setValues([rowData]); } else { sheet.appendRow(rowData); } return ContentService.createTextOutput(JSON.stringify({ result: "success" })).setMimeType(ContentService.MimeType.JSON); }
